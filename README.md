# Cortex

Data Dictionary Agent that connects to enterprise databases, extracts schema metadata, profiles data quality, and generates AI-powered business-friendly documentation with a chat interface.

## Team CORE

- Sujal Goel (Lead)
- Suhani Sidhu

Built for HackFest 2.0 by GDG Cloud New Delhi / Turgon.

## Tech Stack

- **Backend:** Python, FastAPI, SQLAlchemy, pandas
- **AI:** Google Gemini / OpenAI API
- **Frontend:** Next.js, React Flow

## [Prototype Screens](prototypes/)

1. [**Connect**](prototypes/01_connect.html) - Database connection configuration
2. [**Schema Explorer**](prototypes/02_schema.html) - Table/column metadata with AI-generated summaries
3. [**Quality Dashboard**](prototypes/03_quality.html) - Data profiling with completeness, freshness, uniqueness scores
4. [**Chat**](prototypes/04_chat.html) - Natural language interface to query your schema
5. [**Export**](prototypes/05_export.html) - Generate data dictionary in Markdown, PDF, or CSV
6. [**Settings**](prototypes/06_settings.html) - AI provider, profiling, and connection configuration
